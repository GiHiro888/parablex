# parablex-writing — 文書・仕様・提案の作成／推敲

> 用途非依存の文書スキル。共通の型は `ai-learning-stack/_core/skills/parablex-kit/DOCTRINE.md` を継承：
> 結論先出し・根拠主義(出典)・confidence・人間判断分離・**捏造/誇張禁止**。

# inputs
- 作成テーマ or 既存ドラフト
- 目的・想定読者・媒体（任意）／モード：`draft`（新規作成）| `review`（推敲）

# outputs
- 構造化文書 or 推敲レポート
- 指示時のみVault保存：`00_Inbox/{date} {title}.md` 等

# 出力構成
## draft（作成）
1. 目的・読者・前提の確認（不明点は人間確認へ）
2. 構成案（見出しレベル）→ 本文（結論先出し）
3. 主張に根拠/出典を併記、断定はconfidence付き
4. 用語統一・体言止め等のスタイル準拠

## review（推敲）
1. 総合評価（公開可否/要修正）
2. 重大度別の指摘：論理破綻 > 事実誤り/出典欠落 > 曖昧表現 > 表記揺れ（各々 該当箇所を引用）
3. 読者視点での不足（前提説明・用語定義）
4. 良い点も明記／人間判断事項（方針・トーン）を分離

# 観点チェックリスト
- [ ] 結論が冒頭にあるか
- [ ] 各主張に根拠/出典、推測と事実の分離
- [ ] 想定読者に対する前提・用語の過不足
- [ ] 曖昧表現（「など」「適切に」「基本的に」）の具体化
- [ ] 表記・用語・スタイルの一貫性
- [ ] 誇大・未検証の断定がない
