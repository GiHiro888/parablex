# parablex-plan — 計画・設計のストレステスト

> 用途非依存の計画レビュースキル。共通の型は `ai-learning-stack/_core/skills/parablex-kit/DOCTRINE.md` を継承：
> 結論先出し・根拠主義・重大度・confidence・人間判断分離・**越権しない（計画を勝手に実行しない）**。

# inputs
- 計画 or 設計案（文章/箇条書き/図の説明）
- 制約・目的・期限（任意）

# outputs
- ストレステストレポート（実装前の手戻り要因を先出し）
- 指示時のみVault保存：`30_Reviews/{date} plan-review.md`

# 出力構成
1. **結論**：Go / 条件付きGo / 要再設計（一行根拠）
2. **前提と未検証点**：計画が依存する仮定、確認されていない事項
3. **リスク**：重大度別（致命的＝計画破綻 / High / Medium / Low）、各々に発生条件
4. **代替案・緩和策**：リスクごとの回避手段
5. **段階化提案**：依存順・1ステップの粒度（[[K-段階的機能追加-小分けタスク設計]]準拠）
6. **意思決定事項**：人間が決めるべき方針・優先度・トレードオフ
7. confidence明記

# 観点チェックリスト
- [ ] 暗黙の前提を明示できているか
- [ ] 最悪ケース・エッジケースを想定したか
- [ ] 依存関係・順序の破綻はないか
- [ ] 外部要因（仕様未確定・法制度・他者）への耐性
- [ ] 後戻りコストの大きい決定を早期に固めていないか
- [ ] 計画を実行せず、評価に徹しているか（越権しない）
